Название проекта: PORO (Junior Edition)
Жанр: Пошаговый roguelike / RPG (Текстовый квест).
Целевая платформа: Windows (также поддерживается сборка под Linux и MAC).
Целевая аудитория: Начинающие игроки, любители пошаговых стратегий и текстовых квестов.
Описание концепции:
Игрок управляет героем (@), который перемещается по подземельям, состоящим из 15 уровней. На каждом уровне его ждет босс (B) и сундук с сокровищами (C).
Главная механика — это решение загадки (в виде текстового ввода) для открытия сундука и сражение с боссом в пошаговом режиме. 
После победы над боссом игрок попадает в магазин, где может потратить накопленные монеты на улучшение характеристик. Для сражений предусмотрены механики критического удара, брони и уворота.
Уникальные особенности:
«Атмосфера»: Встроенный плеер для фонового воспроизведения музыки на каждом уровне (зависит от файлов в папке music/).
Вариативность: Прокачка персонажа выбирается самим игроком в магазине и при повышении уровня (здоровье, урон, зелья, крит, уворот).
